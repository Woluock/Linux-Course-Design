# 鲁班猫人脸识别系统
基于野火鲁班猫1（RK3566）构建的智能考勤系统

## 实现功能
实现了一个运行于野火鲁班猫1开发板上的人脸识别考勤系统.系统通过USB摄像头实时捕获图像，进行人脸检测与识别，并将考勤结果（含识别到的人名、时间戳、照片）通过局域网传输至个人电脑的Ubuntu虚拟机上，通过GUI界面进行显示和记录。

## 使用声明
client文件夹放到鲁班猫，server文件夹放到Ubuntu系统中。  

通过网线接口连接鲁班猫与个人电脑，确保Ubuntu系统与鲁班猫在同一个局域网中  

检查需要安装的依赖库是否全部安装。  

查询鲁班猫的IP地址,将  
client/GUI.py, client/tcp_client.py, server/GUI.py, server/tcp_server.py  
中的'192.168.137.96'全部替换为鲁班猫的IP地址。  

在鲁班猫和Ubuntu系统中同时运行main.py，依照GUI界面中的提示进行操作。
